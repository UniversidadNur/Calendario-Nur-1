# Calendario 2026 - Universidad Nur

Página web estática (sin dependencias) que muestra el calendario completo de 2026 y resalta fechas importantes.

## Abrir

- Abre `index.html` con tu navegador.

## Eventos

Cada evento inicia el día anterior a las 18:00 y finaliza el día indicado a las 18:00.

Los eventos se configuran en `app.js`.
