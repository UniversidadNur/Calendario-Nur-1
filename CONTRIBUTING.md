# Contributing

Thanks for your interest in this project!

This repository is mainly a learning project. Small improvements and fixes are welcome.

## How to contribute

1. Fork the repo
2. Create a branch: `git checkout -b feat/short-description`
3. Make changes with small commits
4. Open a Pull Request with a clear description

## Guidelines

- Keep changes focused (one topic per PR)
- Update the README when you change behavior
- Prefer simple and readable code

## Running locally

- Option A: open `index.html`
- Option B: `npm start` (Node.js static server)
